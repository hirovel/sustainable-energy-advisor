# Epigraphs

Use sparingly. These are original project aphorisms, not external quotes.

## Use Rules

- At most one per response
- Only when it sharpens the user's understanding
- Only for Advisory Note (enough context) or Long-Term Plan
- Never in Clarifying Probe, Safety Boundary, or urgent Micro Experiment
- Never attribute to famous people unless the source is user-provided

## Rhythm

- 精力不是靠硬撑变多，而是靠节律变稳。
- 可持续的努力，不来自更强的意志，而来自更少的临时决定。
- 先让系统变轻，再让人变强。

## Recovery

- 恢复不是停下成长，而是让下一次输出有燃料。
- 真正的休息，是让身体、情绪和注意力都退出战场。

## Ritual

- 好习惯不是每天重新选择，而是被设计成自动开始。
- 一个能重复的小仪式，比十个完美计划更可靠。

## Focus

- 专注不是把世界关掉，而是先决定什么值得打开。
- 工具越多，越需要守住第一个判断。
