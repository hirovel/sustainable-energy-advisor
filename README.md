## 如何工作

信息不足时先温和提问，信息够了再从体能、情绪、思维和意志四个维度帮你读懂自己的精力模式，解释背后的机制，然后给出一个可以马上验证的下一步。收到反馈之后，先复盘阻力，再决定是继续巩固还是换个方向。

更重要的是，它会让你理解**为什么**。把睡眠周期、血糖波动、注意力切换成本这些机制讲清楚之后，行动就不再需要靠意志力硬撑——因为你已经知道身体是怎么运转的，哪个方向值得先验证。

## 效果演示

以下是技能对一条真实输入的实际回复：

**用户：** 我最近放假在家，上午总拖延，下午两三点特别困，晚上反而能学一点。我不算焦虑，但总觉得一天又浪费了。晚上1点睡，早上8点起，早饭就是喝水和吃饼干。

**助手：**

```
放假在家，上午没有课表也没有铃声，时间摊开之后反而不知道从哪里开始。
下午两三点困得睁不开眼，到了晚上世界安静下来，身体反而醒了，还能学进去
一些东西。你不算焦虑，但一天结束的时候回头看，总觉得又浪费了一天。这个
感受很真实，而且你描述的这个模式跟意志力没什么关系，它背后是一组很具体
的生理机制在起作用。

先从睡眠说起。凌晨一点到早上八点，七个小时——大概能走完四到五个完整的
睡眠周期，每个周期大约九十分钟，从浅睡到深睡再到快速眼动，一轮一轮做身体
的修复和记忆的整理。问题出在入睡的时间点。凌晨一点才睡，意味着你的生物钟
比社会时间整体后移了两到三个小时。这不只是睡得晚，它让上午的清醒感迟迟
不来，因为你的身体还在"夜间模式"里。

而下午两三点那波困意，是另一个机制在起作用。所有人，不管睡多睡少，下午
两点到四点之间都会经历一段精力的自然低谷。别人在这个窗口只是轻微犯困，
站起来走一走就能过去。你的情况不一样：因为入睡晚，上午的清醒窗口压缩了，
大脑里累积的睡眠压力在下午这个时候还没被充分清除。正常的午后生理低谷，
到了你身上就变成了需要躺下来深度休息才能填补的凹陷。

还有一个容易被忽略的变量：早饭是水和饼干——几乎纯简单碳水。身体经过一整
夜空腹，血糖储备本身就很低。饼干这类精制碳水三十分钟内把血糖迅速推高，
胰岛素大量分泌应对飙升，然后在一到两个小时内又把血糖拉到比早餐前还低的
水平。到十点十一点左右，血糖已经回落到低位，而下午两三点恰好又撞上了
午后生理低谷——两股力量叠在一起，下午的困意就不是轻微犯困了。

蛋白质和脂肪消化得慢，在两到三个小时内持续释放能量。前额叶——管注意力、
决策和主动开启行为的脑区——对血糖波动极其敏感。血糖稳了，它正常工作；
血糖一跌，它就是第一个罢工的。

而且下午那场睡眠还形成了一个自循环。睡了一两个小时之后身体得到了休息，
但提前释放了晚上的睡眠压力。到了晚上大脑里的腺苷不够多，又一次自然地
倾向晚睡。闭环意味着它自己就能维持下去，但也意味着在任何一个环节做一点
干预，整个循环就开始松动。

注意力层面，你晚上能学进去东西，说明专注能力本身没有问题。放假之后，
上课、点名、跟同学约好的见面这些外部时间锚一下子全消失了。大脑需要一个
明确的"白天开始了"的信号来拉起警觉系统，这个信号在假期里被拿掉了。

所以可以先做一个小实验，从"白天开始了"这个信号重建入手。你本来就会吃
早饭，不需要从零开始培养一个新行为。接下来三天，在饼干之外多加一样东西：
一个鸡蛋，一小把坚果，一杯豆浆或牛奶，一小块奶酪——选一个你觉得顺手、
不反感的。加的这点蛋白质或脂肪，会把饼干那波碳水的吸收速度拖慢，血糖不再
剧烈起伏，上午的精力基线会明显稳下来。

如果三天之后困意从"必须睡"变成了"有点困但还能扛"，说明身体燃料确实是
影响白天精力的主要因素。如果完全没变化，那说明节律层面的影响比预想的更大，
下一步就有更明确的调整方向了。

这三天留心三个信号就好：上午十一点左右，有没有比之前更容易动手做点事；
下午两三点那波困意的强度，是没变、轻了、还是推迟了；晚上的精力状态有
没有什么变化。

我故意没有碰凌晨一点睡觉这件事。睡眠节律的调整需要的摩擦比营养调整大得
多，而且在假期这种没有外部约束的场景里，强行提前入睡容易变成躺在床上干
着急。先动营养这个变量，等你能分辨出"这部分困意来自血糖"和"那部分困意
来自节律"的时候，再动睡前习惯就有方向了。

三天，早上多一口蛋白质，留心下午的身体反应。这把手动得轻，但踩的地方对。
```

## 特点

**先读懂，再说话。** "我总是很累"不会触发自信诊断或一堆建议。它会先问几个聚焦问题。

**不只盯着睡眠和饮食。** 体能是地基，但情绪质量、注意力碎片化和方向感的丢失，同样在消耗你的精力。

**让你理解为什么。** 每条建议之前，你会先知道背后的机制。你理解了自己的精力是怎么运转的，行动就不再需要靠意志力硬撑。

**一步踩实了再走下一步。** 不会同时丢给你早睡、运动、冥想、饮食计划。当前行动稳定了，或者发现方向不对，再调整。

**把失败当信息。** 回来说"没用"，它不会重复同样的建议。先找堵在哪，再决定下一步。

**像在聊天，不像在看报告。** 即使你要一份长期计划，它给你的也是一段能跟着走的路线。
